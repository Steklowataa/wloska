This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Dependencies

Next.js
 – React framework do SSR/SSG i budowy aplikacji fullstack

React
 – biblioteka do tworzenia interfejsów użytkownika

React DOM
 – integracja Reacta z DOM

React Hook Form
 – zarządzanie formularzami i walidacją

@hookform/resolvers
 – integracja formularzy z bibliotekami do walidacji (np. Zod)

Zod
 – walidacja schematów danych

Validator.js
 – dodatkowe funkcje walidacji stringów (np. email, URL)

Framer Motion
 – animacje dla Reacta

Lucide React
 – zestaw ikon dla Reacta

React Icons
 – zestaw ikon z wielu bibliotek

React Spinners
 – animowane spinnery ładowania

Scroll Into View
 – płynne przewijanie do elementu

Tailwind CSS
 – narzędzie do stylowania w oparciu o utility classes

@tailwindcss/postcss
 – integracja Tailwinda z PostCSS

Tailwind Merge
 – inteligentne łączenie klas Tailwind

Clsx
 – warunkowe łączenie klas CSS

@radix-ui/react-label
 – komponent Label z Radix UI

@radix-ui/react-slot
 – util do komponowania komponentów w Radix UI

@next/font
 – optymalizacja fontów w Next.js
